GAMEBOY ADVANCE GAMES

I developed two games in C using a GameBoy Advance emulator. This was my first time developing in C and working in Ubuntu. Each game was inspired by some of my favorite games growing up.

PROJECT OVERVIEW

Q. How long?
A. 1 week each

Q. With whom?
A. Individual Project

Q. Using what?
A. C, Ubuntu, GBA Emulator

WORD'S HARDEST GAME: FIRE EMBLEM EDITION

The purpose of this game is to try to suicide run across the screen and not get hit by the enemies that are blocking your way. After you make it across, you have to defeat the final commander that is sitting on a fire breathing dragon.
